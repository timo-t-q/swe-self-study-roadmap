# SWE Self-Study Blueprint

Professional, single-page roadmap for learning Software Engineering without a formal degree.

This project presents a structured path from zero to job-ready SWE skills, including:
- learning phases and sequence,
- practical project milestones,
- curated learning resources,
- job role targets with salary ranges,
- readiness checklist for applications.

## Live Repository

- GitHub: https://github.com/timo-t-q/swe-self-study-roadmap

## Project Goals

- Turn a traditional SWE curriculum into a self-study execution plan
- Keep content practical and hiring-focused
- Provide clear progression from fundamentals to employability
- Offer realistic role and compensation visibility for learners

## What Is Included

- `index.html`
  - Fully self-contained page (HTML, CSS, JS)
  - Responsive layout for desktop and mobile
  - Professional visual system with section-based navigation
  - Interactive readiness checklist with progress tracking

## Who This Is For

- Self-taught developers
- Bootcamp learners wanting stronger structure
- Students planning an SWE career outside formal degree programs
- Career switchers building a clear technical roadmap

## Core Sections

- Learning Path
- Resources
- Projects
- Jobs & Salary
- Readiness
- Mentor Notes

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Syne, Manrope, IBM Plex Mono)

## Run Locally

No build process required.

1. Clone the repository
2. Open `index.html` in your browser

Optional local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Design Direction

- Dark, premium interface
- Clear visual hierarchy for readability
- Card-based modular information architecture
- Subtle motion and progressive disclosure via tabs
- Mobile-first responsive behavior

## Notes

- Salary numbers are directional ranges, not guarantees
- Hiring outcomes depend on geography, specialization, portfolio quality, and interview performance
